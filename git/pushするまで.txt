#コマンドでgitでpushするまで


1.githubのアカウントを作成


2.SSH KEYの作成 


3公開鍵の登録





コマンド 

 git add .
 git commit -m "コメント"
 git remote add origin git@github.com:ユーザー名/リポジトリ名.git
 git push origin master
